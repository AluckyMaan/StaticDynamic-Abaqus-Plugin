# StaticDynamic v0.2.0

This release focuses on layered-soil viscous-spring boundary conversion and external geostatic balance inputs.

## Highlights

- Layered soil boundary grouping by adjacent element material.
- Interface boundary nodes split by adjacent material contribution.
- ODB and CSV geostatic reaction inputs.
- ODB final-frame displacement balance check with default tolerance `1.0e-4`.
- `BoundaryInfo.csv` export for boundary review and external RF table generation.
- `X / Y / Z` vertical-axis selection and automatic viewport orientation.
- Chinese documentation page: [README.zh-CN.md](../../README.zh-CN.md).

## Geostatic Input Formats

ODB input reads boundary-node `RF` from the final frame of the selected step and checks final-frame `U`.

CSV input uses:

```csv
nodeLabel,RF1,RF2,RF3
1,-441759.12,441759.12,1293986.8
2,-441759.12,-441759.12,1293986.8
```
